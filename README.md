# Steam Player Engagement & Churn Analysis

## Project Overview

This project analyzes player engagement and lifecycle behavior on Steam.  
The main goal is to understand which factors are associated with long-term activity and which players are more likely to churn.

The analysis focuses on player activity, owned games, achievements and social connections.

## Business Questions

1. What factors are associated with longer player activity?
2. Do players with more friends stay active longer?
3. Do players with more owned games have lower churn risk?
4. Are achievement hunters more engaged?
5. Can we predict which players are likely to churn?

## Data Storage

The full raw dataset is stored locally in a PostgreSQL database and is not committed to this repository due to file size.

The repository contains only:
- analysis notebooks,
- SQL transformation scripts,
- documentation,
- generated metadata reports,
- dashboard files or screenshots.

To reproduce the analysis, users should download the source dataset and load it into PostgreSQL.
